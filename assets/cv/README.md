# PDF CV Placeholder

This directory should contain:
- `Andrey-Maulana-CV.pdf` - Your professional CV in PDF format

## How to Create the PDF:

1. Use the content outline in `CV-PDF-Content-Outline.md` (in root directory)
2. Design the PDF using:
   - Canva (easiest, templates available)
   - Figma (more control)
   - Adobe InDesign (professional)
   - Microsoft Word/Google Docs (simple)

3. Follow the design guidelines:
   - Color: #0066ff (Electric Blue) as primary
   - Font: Poppins for headers, Inter for body
   - 4 pages maximum
   - ATS-friendly (no complex graphics)
   - File size < 2MB

4. Save as `Andrey-Maulana-CV.pdf` in this directory

5. Test the download button on the website

## Temporary Solution:

Until you create the PDF, the download button will show a 404 error. You can:
- Remove the download button temporarily
- Create a simple PDF placeholder
- Or create the full PDF now using the outline provided
