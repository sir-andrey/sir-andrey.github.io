# Assets Images - Placeholder Guide

## File yang Perlu Disiapkan:

### 1. Profile Photo
**Filename:** `profile.jpg` atau `profile.png`
- **Ukuran:** 400x400px (square)
- **Format:** JPG atau PNG
- **Deskripsi:** Foto profil profesional dengan background bersih

### 2. Company/Project Logos (Opsional)
**Folder:** `assets/images/companies/`
- `unictive-logo.png` - Logo PT. Uniktif Media Indonesia
- `nine-operation-logo.png` - Logo CV. Nine Operation Development
- `module-intracs-logo.png` - Logo PT. Module Intracs Yasatama
- `sangkuriang-logo.png` - Logo PT. Sangkuriang Internasional

**Ukuran:** 200x80px (landscape) atau 100x100px (square)
**Format:** PNG dengan background transparan

### 3. Project Screenshots (Opsional)
**Folder:** `assets/images/projects/`
- Format: `project-name.jpg` atau `project-name.png`
- Ukuran: 800x600px atau 1200x800px
- Contoh:
  - `asean-summit-2023.jpg`
  - `bca-wealth-summit.jpg`
  - `komdigi-aduan.jpg`
  - dll.

### 4. Tech Stack Icons (Opsional - bisa pakai CDN)
**Folder:** `assets/images/tech/`
- `laravel.svg`
- `php.svg`
- `mysql.svg`
- `redis.svg`
- dll.

**Note:** Untuk tech icons, lebih baik pakai dari CDN seperti:
- https://cdn.jsdelivr.net/gh/devicons/devicon/
- https://simpleicons.org/

### 5. Background/Decorative Images (Opsional)
**Folder:** `assets/images/bg/`
- `hero-bg.jpg` - Background untuk hero section
- `pattern.svg` - Pattern untuk dekorasi

---

## Cara Menggunakan:

1. **Siapkan gambar sesuai spesifikasi di atas**
2. **Rename file sesuai dengan nama yang tertera**
3. **Copy ke folder yang sesuai**
4. **Gambar akan otomatis muncul di CV HTML**

---

## Placeholder Sementara:

Untuk sementara, CV akan menggunakan:
- **Profile photo:** Inisial dengan background gradient
- **Company logos:** Text-based logo
- **Project images:** Gradient placeholder
- **Tech icons:** Text badges atau CDN icons

Kamu bisa timpa file-file ini kapan saja tanpa perlu edit HTML.
